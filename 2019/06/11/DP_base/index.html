发表于 2019-06-11 | 分类于 算法

做题板子

  • 解答模板
    • 定义子问题
    • 定义目标
    • 写递归式与初始化条件
    • 伪代码
    • 分析时间复杂度
  • 1维子问题：
    • 输入：$x[1…n]$序列，字符串，数组
    • 子问题：$x[1…i]/x[i…n]$前缀或后缀
    • 复杂度：$\Theta(n)$
    • 应用：
      • 杆切割？
      • 最长递增子序列？
      • 上限子数组？

  • 2维子问题

    • 输入：$x[1…n],y[1…m]$
      • 子问题：$x[1…i],y[1…j]$
      • 复杂度：$\Theta(mn)$
      • 应用：编辑距离，最长公共子序列
    • 输入：$x[1…n]$
      • 子问题：$x[i…j]$
      • 复杂度：$\Theta(n^2)$(不一定)
      • 应用：最优二叉查找树,矩阵连乘

  • 3维子问题

    • $d(i, j, k) = min (d(i, j, k − 1), d(i, k, k − 1) + d(k, j, k − 1))$

  • 图上的DP：

    • 在DAG上：拓扑序后转换为1D问题（DAG最短路）
    • 其他树：左右子树递归

  • 背包问题：取/不取的问题
